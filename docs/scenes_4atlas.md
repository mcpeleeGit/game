# 4가지 씬 아틀라스 (scenes_4atlas.png)

`assets/scenes_4atlas.png`는 아래 4개 씬을 **2×2 그리드**로 합쳐 둔 이미지입니다.

| 위치       | 씬/용도           | 설명                         |
|-----------|-------------------|------------------------------|
| **좌상단** | 숲 / 보물         | 나무, 폭포, 해골, 보물상자, 기사 |
| **우상단** | 사막 / 유적       | 사막, 오아시스, 사원, 불덩이, 방패 든 기사 |
| **좌하단** | 마법숲 / 고블린전 | 어두운 숲, 기사 vs 고블린, 데미지 숫자 |
| **우하단** | 화산 / 드래곤전   | 용암, 붉은 드래곤, 불꽃 브레스, 방패 든 기사 |

## Godot에서 사용

- **한 칸만 쓰기:** `Sprite2D`에 Texture로 이 PNG를 넣은 뒤, **Region**을 켜고 `Rect2(0, 0, 절반폭, 절반높이)` 등으로 해당 사분면만 잘라서 사용.
- **아틀라스로 쓰기:** 리소스에서 **New AtlasTexture** 만들고, Atlas = 이 이미지, Region = 해당 사분면 사각형으로 설정 후 `Sprite2D.texture`에 지정.
- **전체 보기:** Texture 그대로 넣으면 4칸이 한 번에 보임 (배경/미리보기용).

이미지가 정확히 4등분이라면, 폭/높이의 절반씩으로 나누면 됩니다 (예: 512×512 → 한 칸 256×256).

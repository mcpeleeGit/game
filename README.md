## 📁 폴더 구조
- `project.godot` : Godot 프로젝트 설정
- `assets/`      : 이미지, 사운드, 맵 등 게임 리소스
- `src/`         : 씬(.tscn), 스크립트(.gd)
- `docs/`        : 기획/설계 문서
- `tools/`       : 맵툴, 에디터 스크립트

## 🚀 실행 방법
1. [Godot 4](https://godotengine.org/download) 설치
2. Godot 실행 후 **프로젝트 가져오기** → 이 폴더(`game`) 선택
3. 에디터에서 **F5** 또는 상단 **재생** 버튼으로 실행

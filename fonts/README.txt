CJK 폰트 설정 (한/중/일 문자 표시)

1. Noto Sans CJK 다운로드
   - https://github.com/googlefonts/noto-cjk/releases
   - NotoSansCJK-Regular.ttc (또는 언어별 NotoSansKR, NotoSansJP, NotoSansSC)
   - 이 폴더에 NotoSansCJK-Regular.ttc 복사

2. OFL.txt
   - 폰트 배포 시 라이선스 파일 포함 (이미 포함됨)
   - 크레딧: "Font: Noto Sans CJK © Google, SIL Open Font License 1.1"

3. 프로젝트 테마
   - theme/default_theme.tres 가 이 폰트를 기본으로 사용
   - Project Settings > GUI > Theme > Custom 에 연결됨
   - 폰트 추가 후 Godot 재시작 또는 프로젝트 다시 열기

# 다국어(i18n) 키 규칙

- **키는 UI에 직접 노출하지 않음.** 항상 `tr("KEY")` 사용.
- **새 텍스트는 반드시 `strings.csv`에 추가** 후 코드에서 `tr("KEY")` 사용.

## 키 접두사 추천

| Prefix   | 용도           |
|----------|----------------|
| TITLE_*  | 타이틀 화면    |
| BTN_*    | 버튼 문구      |
| HUD_*    | HP, Gold 등 HUD |
| MAP_*    | 월드맵 라벨/힌트 |
| REGION_* | 지역 전용      |
| BATTLE_* | 전투 UI/메시지 |
| PROLOGUE_* | 프롤로그     |
| REWARD_* | 보상 문구      |
| LOCKED_* | 잠금 메시지    |
| SETTINGS_* | 설정 화면   |
| LANG_*   | 언어 선택 옵션  |

- ❌ "OK", "YES" 같은 범용 단어만 단독 키로 쓰지 않기  
- ✅ "BTN_OK", "BTN_CONFIRM" 처럼 용도가 드러나게 명명

## 포맷 문자열

숫자/날짜/퍼센트가 들어가면 포맷도 번역으로 두는 것이 좋음.

예: `HUD_GOLD_FMT` → `Gold: %d` / `ゴールド: %d` / `金币：%d`
